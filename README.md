Place the files extracted from the source in the following manner:

'''
data/
└── raw/
    ├── chest-xray/
    │   ├── test/
    │   │   ├── NORMAL/
    │   │   └── PNEUMONIA/
    │   ├── train/
    │   │   ├── NORMAL/
    │   │   └── PNEUMONIA/
    │   └── val/
    │       ├── NORMAL/
    │       └── PNEUMONIA/
    ├── test/
    │   ├── NORMAL/
    │   └── PNEUMONIA/
    ├── train/
    │   ├── NORMAL/
    │   └── PNEUMONIA/
    └── val/
        ├── NORMAL/
        └── PNEUMONIA/
'''
